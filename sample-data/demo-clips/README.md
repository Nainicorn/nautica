# Demo Clips

Place maritime video clips here for dashboard demos and testing.

## Recommended Sources
- Maritime drone footage from YouTube (search "drone ship tracking", "aerial harbor footage")
- Coast guard / public agency surveillance clips
- Port and cargo vessel footage

## Suggested Clips to Curate
- One cargo ship crossing frame
- One fishing boat near coast
- One multi-vessel clip
- One loitering-style clip
- One harbor entry clip

## Formats
MP4, AVI, MOV supported.
